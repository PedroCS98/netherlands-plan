# 🇳🇱 Netherlands Plan

A move-planning website for 4 friends relocating from Portugal to the Netherlands. Tracks phases, personal to-dos, a savings budget calculator, and a FAQ — all in one place.

Personal learning project built during a Full Stack & Web Design course — planned to grow into a multi-page site with authentication and persistent data, and intended as a portfolio piece for tech job applications in the Netherlands.

---

## Features

- **Phase roadmap** — four-phase plan from savings to arrival and setup
- **Per-person to-do lists** — individual action items for Nuno, Rafa, Valter and Otis
- **Budget calculator** — monthly savings table per person with auto-totals (vanilla JS)
- **FAQ section** — accordion-style toggle (vanilla JS)
- **News feed** — placeholder section for Netherlands updates
- **Login form** — to be implemented

---

## Tech Stack

- HTML5 (semantic markup)
- CSS3
- Vanilla JavaScript

No frameworks or libraries — intentionally kept simple while learning the fundamentals.

---

## Design System

- **Fonts** — Playfair Display (headings, 700/800) + Plus Jakarta Sans (body/UI, 400/500/600)
- **Primary colours** — Midnight `#0F1F2E`, Canal `#1B3A5C`, Sky `#7BAFD4`, Dutch Orange `#E8521A`, Tulip Gold `#F5A623`, Parchment `#F5F0E8`, Ink `#2C2C2A`
- **Spacing scale** — 4 / 8 / 12 / 16 / 24 / 32 / 48 / 64px
- **Border** — 0.5px solid, radius 6–10px for components, 99px for pills
- Full style guide saved in `/assets/styleguide_nl.png`
